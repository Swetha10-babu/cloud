An Introduction to the Cloudcomputing concise:

A cloud platform is the operating system and hardware of servers in a data center that are configured to provide cloud computing services to customers.A cloud platform enables businesses to rent access to computing resources on demand over the internet with pay-as-you-go pricing, rather than buying, installing, and managing their own data centers, servers, and software required to have these resources available on premises.



Cloud computing

  • Amazon Web Services: A cloud computing platform that provides businesses and developers with a wide range of IT services over the internet. Businesses can use AWS to host websites, store data, run applications, and perform data analysis.

  • Microsoft Azure: Azure offers a range of cloud services, including computing, analytics, storage, networking, and AI. Customers choose from these services to develop and scale new applications, or run their existing applications, in the cloud.

  • Google cloud Platform:Google Cloud Platform (GCP) is like a virtual toolbox that Google provides for businesses and individuals to build, run, and manage all kinds of applications and data online. Think of it as renting powerful computers and tools from Google, rather than buying and managing your own.
